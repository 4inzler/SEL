SEL Windows Compatibility - File Summary
=========================================

INSTALLATION SCRIPTS:
---------------------
install_sel.ps1              - PowerShell installer (checks Python, installs Poetry, sets up deps)
start_sel.bat                - Batch file launcher (loads .env, starts HIM + bot)
check_system.bat             - System requirements checker (verifies readiness)

EXECUTABLE LAUNCHER:
--------------------
windows_launcher.py          - Python source for all-in-one executable
sel_windows.spec             - PyInstaller configuration
build_windows_exe.bat        - Windows build script
build_windows_exe.sh         - Linux/WSL build script

USAGE:
------
For end users:
  1. Run: install_sel.ps1
  2. Edit: .env (add tokens)
  3. Run: start_sel.bat

For executable distribution:
  1. Build: build_windows_exe.bat
  2. Distribute: sel_launcher.exe + project_echo/ + .env.example
  3. User runs: sel_launcher.exe

For developers:
  - Manual setup: See WINDOWS_INSTALL.md Method 3
  - Testing: poetry run pytest
  - Building: pyinstaller sel_windows.spec

DISTRIBUTION PACKAGE:
---------------------
Recommended structure:
  SEL-Windows/
  ├── sel_launcher.exe or (install_sel.ps1 + start_sel.bat)
  ├── project_echo/
  ├── agents/
  ├── .env.example
  ├── WINDOWS_README.md → README.md (rename for users)
  ├── WINDOWS_SETUP.md
  └── LICENSE.md

SIZE ESTIMATES:
---------------

DEPENDENCIES:
-------------
Required:
  - Python 3.11+
  - Poetry (auto-installed)
  - DISCORD_BOT_TOKEN
  - OPENROUTER_API_KEY

Optional:

  - Docker Desktop (for container deployment)
